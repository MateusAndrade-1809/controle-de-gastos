# 🚀 Projeto Flask - Meu Primeiro Sistema Web

Este é um projeto desenvolvido com o objetivo de praticar desenvolvimento web utilizando Python com Flask, HTML e CSS.

## 🧠 Sobre o projeto

O sistema consiste em uma aplicação web simples que demonstra a integração entre backend e frontend, permitindo a exibição de páginas dinâmicas.

## 🛠️ Tecnologias utilizadas

- Python
- Flask
- HTML5
- CSS3

## ⚙️ Funcionalidades

- Página inicial renderizada com Flask
- Integração entre Python e HTML
- Estrutura básica de aplicação web
